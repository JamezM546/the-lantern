# The Lantern

A cozy corner for manga discovery. The Lantern is a manga tracker and discovery web app where you can search for titles, save the ones you're interested in, and track what you're reading — all wrapped in a warm dark-violet aesthetic.

This project is built from scratch as a hands-on learning vehicle, with every line written by hand to deeply understand the mechanics of front-end development.

## Features

- **Search** manga by title
- **Browse** and discover new series
- **Save** titles to a personal list
- **Track** your reading progress

*(Some features are in progress — this is an actively developed project.)*

## Tech Stack

- **HTML** — page structure
- **CSS** — styling and layout (flexbox, custom properties for theming)
- **JavaScript** — dynamic rendering and app logic
- **[Jikan API](https://jikan.moe/)** — manga data source

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/JamezM546/the-lantern.git
   ```
2. Open `index.html` in your browser.

That's it — no build step required.
